Commands

Command Description Aliases Example

General Commands

Command Description Aliases Example
~invite Displays the invite link for RuneCord ~join ~invite
~id [channel] Displays your user ID if "channel" is not supplied as a suffix ~myid ~id | ~id channel
~about Displays some information about the bot   ~about
~roll number Rolls a number between 1 and x, a number must be supplied   ~roll 100
~twitch username Displays information about the given streamer. A username must be supplied   ~twitch wittyrs

RuneScape Commands

Command Description Aliases Example
~vos Displays the current Voice of Seren districs.   ~vos
~stats username Grabs the stats of the username supplied. ~hs | ~rs3stats ~stats Witty
~osstats username Grabs the oldschool stats of the username supplied. ~osrsstats ~osstats Witty
~time Displays the current in-game time. ~gametime ~time
~reset Displays how long till the game resets.   ~reset
~price username Displays grand exchange information based on item name.   ~price coal
~cache Displays when the next Guthixian Cache is.   ~cache
~viswax Displays the current viswax combinations.   ~viswax

~bigchin Displays when the next Big Chinchompa will be.   ~bigchin
~invasion skill level Displays how much XP you'd gain in troll invasion if your level was skill level and you finished the entire thing. ~trollinvasion ~invasion 99
~lamp small|med|large|huge skill level Displays how much XP you'd get from a lamp based on skill level and size. ~xplamp ~lamp large 99

Configuration

Key Default Value Description
token   Your bots token Must be entered for the bot to work
app_id 168215284161708032 Your bots application ID This is used to create your OAuth invitation link, so it's wise to update it!
command_prefix ~ Set your own command prefix
mod_command_prefix ) Set your own command prefix for mod commands
non_essential_event_listeners true Turn this off if you don't want to track things like serverNewMember, channelUpdated, userBanned etc.
twitter_api   The twitter API ID for your twitter widget (used for VoS) Must be entered for the bot to work
admin_id 124627910232375296 The bot owner's user ID, used for extreme commands like ')reload'
debug true Allows you to view all the debugging information, turn off for less spam in console
show_warn true Makes you see all the warning, nothing too severe that can break the bot
carbon_key   The key that is given to you when you add your bot to Carbon Leave blank to disable carbon stats
whitelist ["168550901836021761", "110373943822540800"] An array of server id's that will not be removed from the inactive list, no matter what
banned_server_ids [] An array of server id's that the bot is not allowed to join

settings
Server Settings

For use with the )settings command, for server mods. Do )help settings for more information about syntax.

Ban Alerts

A message will be sent to the notification channel and to the user when someone is banned.

enable/disable ban alerts
Name Changes

When a user changes their name a message will be sent to the notification channel.

enable/disable name changes
Name Changes

This message is sent to the default channel when a user joins.

disable welcome
welcome Welcome $USER$!
$USER$ replaces with the user's name
$SERVER$ replaces with the server's name
Delete Commands

Certain commands can delete the activating message to help clean chat.

enable/disable delete commands
Notification Channel

Set where ban alerts and name changes are shown.

notify here
Ignored Channels

When a channel is ignored the bot will not respond to commands unless they are listed under "mod commands".

About RuneCord

RuneCord was made using Discord.js. It is a bot made for RuneScape with many features such as voice of seren, stats, adventures log, price lookup, and much more.
