A general purose Discord bot for RuneScape

Invite me to your Discord

Hosted in the Cloud

The official bot is hosted DigitalOcean. The server uses high-speed internet, and an SSD, which allows for the bot to have speedy response times.

Open-Source

By making the bot open-source, it allows you, the user, to create your own version of RuneCord just for your clan, tailor it to lookup clan stats, make your own custom commands for it, do whatever you wish!

Easy to work with

With very simple and reliable commands, the bot is very easy to work with. Just add the bot to your server, and away you go!

Commands

All normal commands start with ~ | All moderator commands start with ( | Type ~help in Discord to view commands from Discord.


General Commands

Command Description Example Aliases
~invite Gives you an invite link to invite RuneCord to your server. ~invite join
~id channel Returns the user's ID if no suffix is supplied, if channel is supplied for the suffix, will return the channel's ID. ~id | ~id channel myid
~about Tells you various things about the bot. ~about  
~roll [number] Rolls a number between 1 and number. ~roll 100  
~twitch [username] Displays information about the given streamer. ~twitch WittyRS

Configuration

Key Default Value Description
command_prefix ~ Set your own command prefix
mod_command_prefix ) Set your own command prefix for mod commands
non_essential_event_listeners true Turn this off if you don't want to track things like serverNewMember, channelUpdated, userBanned etc.
debug true Allows you to view all the debugging information, turn off for less spam in console
show_warn true Makes you see all the warning, nothing too severe that can break the bot
whitelist ["168550901836021761", "110373943822540800"] An array of server id's that will not be removed from the inactive list, no matter what
banned_server_ids [] An array of server id's that the bot is not allowed to join

Server Settings

For use with the )settings command, for server mods. Do )help settings for more information about syntax.

Ban Alerts

A message will be sent to the notification channel and to the user when someone is banned.

enable/disable ban alerts
Name Changes

When a user changes their name a message will be sent to the notification channel.

enable/disable name changes
Name Changes

This message is sent to the default channel when a user joins.

disable welcome
welcome Welcome $USER$!
$USER$ replaces with the user's name
$SERVER$ replaces with the server's name
Delete Commands

Certain commands can delete the activating message to help clean chat.

enable/disable delete commands
Notification Channel

Set where ban alerts and name changes are shown.

notify here
Ignored Channels

When a channel is ignored the bot will not respond to commands unless they are listed under "mod commands".

)ignore
)unignore
